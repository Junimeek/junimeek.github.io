Baldi's Extended Schoolhouse

Update 0.5.2 Progress

90%
100%

Beta 0.5.2 will be a relatively small update, however, Beta 0.6 will bring a major change to the structure of save files. Most of the code is done and will be in 0.5.2 to allow players to upgrade their save files before updating to 0.6. Once you upgrade your save files to the new system, they will become completely incompatible with all versions before 0.6. If you care about being able to go back to older versions of the game, make a backup of the current save files (there's four files you need to copy).

In addition to the new save file structure, the save file directory will also be changed. Currently, it is in AppData/LocalLow/JuniDev, and it'll be changed to AppData/LocalLow/Kyokura (on Windows, anyway. I don't know where the files are stored on Linux, so if you use Linux you're just gonna have to figure it out like you do everything else about your computer).
