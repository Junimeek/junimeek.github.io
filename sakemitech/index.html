Baldi's Extended Schoolhouse

IMPORTANT NOTICE:
If you downloaded Baldi's Extended Schoolhouse before October 8, 2025, please be aware that it shipped with a security vulnerability that was discovered in all versions of Unity Engine from 2017-onwards. A patch is available below. It should also be noted that the vulnerability exists in vanilla Baldi's Basics Classic too, though mystman12 has already announced that he will fix the issue in all major releases of Baldi's Basics. You can read more about it here.

It's Baldi's Basics, but with a simple twist: somehow, Baldi has raised enough money to expand the schoolhouse! And of course, more space means more classroom, which means more noteboos for your friend to forget. Like usual, find all the notebooks to win! It's THAT easy!

This mod is for Windows (or as some of you weirdos call it, "PC") and Linux platforms only (i.e., no mobile build. please don't ask for one).
