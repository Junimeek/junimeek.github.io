Baldi's Basics Classic Decompilation

An important update regarding the decomp:
The decompilation is now back online at its original url. This page will remain active for historical purposes, but you should be downloading the decomp from this website instead.

Recently, the most widely-used decompile of Baldi's Basics Classic was deleted for some reason. In response to this, for the sake of preservation I found the unmodified copy that I had saved in my downloads folder and am making it publicly available.

A quick note about the included features

Please note that this is NOT the newest version of the decompile that was ever released; the newest version had extra debugging features and other quality-of-life improvements. While I would love to be able to provide that version for you, I simply do not have it. I never had any need to download a newer version of the decomp since I had already overhauled a ton of code in my own mod anyway. As such, I am only able to provide the releases from mid-2024 since those are all that I have saved. This does unfortunately mean that there is no controller support since at the time, the decompile still needed Rewired to utilize controller input, and thus it was not included.

Downloads

I'll work on proper installation instructions at some point since I am rushing to get this page out as quickly as possible, but for now, you will need to install a copy of Unity Engine version 2019.2.8f1 or newer. However, remember that a security vulnerability was discovered in Unity Engine last year, and as such, you should instead install 2019.4.41f1 or anything newer than that. Upgrading the project to newer engine versions should be relatively straightforward, though.
