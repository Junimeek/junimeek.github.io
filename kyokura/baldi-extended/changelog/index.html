Version 0.5.2

Release date: December 3, 2025

This is a small update, but a very important one: Version 0.6 will change how save files are structured, and this version will allow players to upgrade their save files to the system that 0.6 will be using. While 0.6 will be able to detect old save files, it will not be able to detect the settings used in previous versions, hence why this version will allow you to export your settings.

Gameplay changes

  • Gotta Sweep's hitbox is now disabled when using a Party Popper and while a party is active
  • Decreased the maximum number of projectiles available at the same time in the Null boss battle from 4 to 3

Miscellaneous changes

  • Reorganized the layout of the launcher
  • The new update message is now a banner at the top of the launcher's screen
  • Changed the credits in the launcher and in-game credits menu to reflect PSPLeafFox's username change
  • Made it so double clicking the Quit button on the menu will force the game to close immediately
  • The background music (ie, "Schoolhouse Trouble" and "Schoolhouse Escape (Free Run)") has now been replaced with a revolving ensemble of four songs
  • The Null boss fight now has a health bar

Bug fixes

  • Fixed the flashlight not working at all. How I missed this one is beyond me.
  • Fixed the Crystal Zesty Bar not playing a sound when eating one
  • Fixed a bug where the second exit would not play a sound in Safe Mode
  • Fixed a bug where using the Party Popper within 13 seconds of reaching the first exit would cause the background music to stop working
  • Fixed an issue where using the Party Popper would not stop the normal background music in endless mode
  • Fixed an issue where projectiles in the Classic Dark challenge kept spawning on top of eachother
  • Fixed a tiny issue where dancing Baldi's sprites were mirrored

Internal changes

  • Removed Newtonsoft JSON, Fluid MIDI Player, and some unnecessary files that I left in in the StreamingAssets folder
  • Moved some of the scripts for Dark Mode to a separate ChallengeController script

Version 0.5.1

Release date: October 28, 2025

This is the update I spent a whole year on without putting in anything that tells you that I spent a year on it.

Gameplay changes

  • The addition/subtraction problems when Difficult Math is enabled now have three digits
  • You will now automatically equip a flashlight during the finale to improve navigation
  • Changed the credits in the launcher and in-game credits menu to reflect PSPLeafFox's username change

Miscellaneous changes

  • Changed the developer name to Kyokura. Note that internally, "JuniDev" will continue to be what the company name is to ensure compatibility with existing save files.
  • The notebook count display from BB+ has been added to this mod
  • Items and money will now play a sound when being picked up
  • You can now see the center UI hand icon during the Null bossfight
  • Removed the bus that was in the Dark Mode challenge map. seriously why was it even there
  • Added a few new posters
  • Removed a few posters for personal reasons

Bug fixes

  • The random ambience noises are now properly filtered through the SFX volume control
  • The sprite bouncing effect on pickups (notebooks and items) is now consistent regardless of framerate
  • Fixed a bug where the school colors would suddenly flip to red when reaching the second-to-last exit in safe mode
  • Fixed an issue where some characters would not appear on the quick map
  • Fixed an issue where you could see parts of the UI when looking out the window of the East Exit after defeating Null

Internal changes

  • Updated the Unity Engine version to 2021.3.45f2 to address a security vulnerability that all applications built in Unity Engine 2017.1-onwards contain. Unity Technologies has advised all developers to update their Unity version asap. You can read more about it here: "Unity Platform Protection: Take Immediate Action to Protect Your Games and Apps"
  • Some variables are now set in the new MasterController gameobject. This was done to make modifying the scenes a little easier since some things would instantly break when making changes to the UI.

Version 0.5.0

Release date: October 19, 2024

Probably the most productive update so far tbh

Gameplay changes

  • New mode: Challenge mode!
    • New challenge: Dark mode! Which is basically just BBCR's Null Style lmao
  • You can now charge your stamina up to 300% if you're standing in an infinite stamina zone!
  • The Baldicator will now queue its expressions for sight, next sound, and wandering. This is to ensure that you will still know what Baldi is doing while other animations are playing.
  • The Baldicator has a different sprite for Sight now for improved visual clarity
  • During the finale, once you're at just 1 exit remaining, the school will get darker as you get closer to the exit

Item changes

  • New item: the Dangerous Hammer! Smash through any window you can find to be able to go through it!
    • For now, this item is only available in the Dark Mode challenge.
  • The duration of the Yellow Door Lock and Big'Ol Boots have been increased to 30 seconds
  • The number of times a door treated with WD-NoSquee can be opened silently has been increased to 6 times
  • A visual indicator has been added for doors treated with WD-NoSquee
  • Eating a zesty bar will now play a sound

Miscellaneous changes

  • There is a new category in the menu: Patches. Right now the only things that are there are selecting between your monitor's refresh rate or unlimited framerate, and fixing the noisy doors bug. More info about the latter is under the Bug Fixes category.
  • The music during the finale has been altered
  • There is a new sequence for the normal ending
  • The visuals of the quick map have been changed
  • Saving has been completely overhauled. You'll see a prompt upon launching the game the first time you open it. The game will have an indicator in the top left corner of the screen when saving is in progress. If you were playing an older version of the game, it should detect that old data and upgrade it to the new system.
  • The Data Management menu actually works like a data management menu now
    • There are now a few more things that will be saved in your high scores: Your best time in Story Mode, and how many of each item you've used in each map
  • A Family Friendly option has been added. This only affects some minor schoolhouse decoration.

Bug fixes

  • The bug where clicking on an open door or simply walking through open swinging doors causing Baldi to be alerted has been fixed. However, I acknowledge that some people may use this bug as part of their strategy. As such, if you prefer this bug being in the game, you can optionally disable this fix in the settings's Patches menu.
  • Clicking on an open door that has been treated with WD-NoSquee will no longer cause it to decrement the number of times the door can be silently opened
  • Fixed the bully not detecting any items on you if your first item slot happens to be empty
  • The coin drop sound from the vending machines are now properly filtered through the SFX volume control

Version 0.4.3

Release date: August 12, 2024

Productivity at its least mediocre

Gameplay changes

  • There's now a quick map! It can be toggled on and off by simply pressing Tab.
    • Upgrades for the map can be purchased in the Principal's office. There is a poster next to the map upgrade vending machine that will explain in further detail.
  • You can now carry an infinite amount of quarters! All quarters will automatically be added to your wallet.
    • To use a quarter, simply look at a vending machine and right click, and it will use any quarters you have collected. If your inventory is full, select the item you want to replace with whatever the vending machine will dispense, and it'll replace that item with the new item.
  • As a result of several new items being added (explained further below), there is now a significant change to the vending machines: They will now cost varying amounts of quarters depending on what item they give you.
    • Diet BSODA and Crystal Zesty Bars cost 1 quarter
    • BSODA and Energy Bars cost 3 quarters
    • Map Upgrades cost 1-4 quarters depending on how many you want to buy
    • These changes do not apply to the Classic map; all items will continue to cost just 1 quarter as per usual.
  • Safe mode will now spawn a Ghost Baldi once you get a question wrong. Ghost Baldi will chase you just like normal Baldi, he just can't end your game.
  • Added two faculty rooms to Juniper Hills
  • Increased the number of exits in Classic Extended to 6
  • Increased the number of exits in Juniper Hills to 5

Item changes

  • Collecting an item while your inventory is full will no longer completely discard an item; instead, it will place the item where you collected the new item, just like in BBCR.
  • The map you play will now determine the number of inventory slots you have to work with:
    • Classic: 3 slots
    • Classic Extended: 5 slots
    • Juniper Hills: 4 slots
  • FOUR new items!
    • Diet BSODA: It's just like regular BSODA, but it only lasts 3 seconds
    • Crystal Flavored Zesty bar: Increases your stamina by 70%
    • Party Popper: Start a party in the cafeteria, the gym, or in any classroom! It can be found in the Principal's office, and it will attract all characters. What fun! It will also be less effective the more notebooks you have (unless you're playing endless mode, where it'll always last for 85 seconds instead).
    • Wallet: The wallet is permanently attached to your inventory and holds coins.
      • This does not apply to the original Classic map; quarters will still be put in your inventory like normal there
  • The effects of the Zesty Bar can now be stacked
  • The Attendance Slip has been buffed slightly: the character you send to attendance will now stay there for the entire game, instead of for just 3 notebooks.
  • The Alarm Clock has been changed to act like it does in BBCR: you can set the time after dropping it somewhere, and it'll make Baldi ignore ALL sounds with the exception of the Party Popper.
  • Added a new quarter that will spawn in a random room
  • Removed a few standing quarters from Juniper Hills due to there being an excessive amount of money on that map. You can now only find a maximum of $2.50 in Classic Extended, and $2.25 in Juniper Hills.

Miscellaneous changes

  • Added a few meme posters in the halls
  • Added a new loading screen
  • The game will tell now you how many exits are remaining in the top left of the screen
  • Added the rest of Baldi's jumpscare sounds introduced in BB+
  • Baldi's AI has been slightly improved to keep him moving instead of constantly getting stuck in place for a few seconds
  • The Baldicator's animation times are slightly quicker now
  • Its a Bully will now actually go to detention if the Principal catches him. Don't worry, he won't steal anything from you while he's in there.

Bug fixes

  • Fixed a bug where the player would not look at the Principal when sent to detention, or at Baldi when Arts and Crafters teleports you. This is actually a bug that has existed in the original game for forever, I just never found a fix for it until now.
  • Fixed a bug where Arts and crafters would softlock the player in Classic Extended and Juniper Hills
  • Fixed a bug where the characters would phase through a wall in Juniper Hills
  • Fixed a bug where picking up an item would sometimes result in you collecting the wrong item
  • Fixed a few bugs with the center item display
  • Fixed an issue that would allow you to use vending machines on the opposite side of a wall
  • The fog during the finale now renders correctly

Version 0.4.1

Release date: July 15, 2024

This version is extremely broken because of two major oversights (Juniper Hills is completely impossible, and Baldi is broken). If you play this version, just keep that in mind.

Gameplay changes

  • Baldi's AI has changed slightly: he will remember more sounds that you make, so be careful where you go, he can follow you a lot easier now!
  • When you open Gotta Sweep's door for the first time, he will immediately begin sweeping

Item changes

  • Collecting an item while your inventory is full will no longer completely discard an item; instead, it will place the item where you collected the new item, just like in BBCR.
  • New item: The Attendance Slip!
    • The attendance slip will spawn in a random faculty room. It will send Playtime, 1st Prize, or Gotta Sweep home (at least, until the Principal notices. He WILL catch you, so use it wisely)!
  • The Speedy Sneakers have been nerfed, as they were EXTREMELY broken before: they will now break much faster.

Miscellaneous changes

  • New Help menu! It explains the story, items, controls, and the Baldicator.
  • Background music volume has been slightly decreased

Version 0.4

Release date: July 9, 2024

I did stuff

Gameplay changes

  • Gameplay style options!
    • Difficult Math: Adds multiplication and division to the math game
    • Safe Mode: Baldi won't chase you, giving you a stress-free experience
  • The Principal will now only be able to see you when you can obviously see him. This is essentially just nerfing his x-ray vision that allowed him to see you through multiple sets of yellow doors.

Miscellaneous changes

  • Added settings for BGM, SFX, and Voice audio
  • Added the ability to reset your data
  • Added new icons that appear in the center of the screen when you can interact with something. For example, doors and picking up items will show a hand, having the quarter selected while looking at a vending machine will show a quarter, etc.
  • There's two different versions of the same secret ending now: one for getting all questions wrong with safe mode turned on, and one without using safe mode.

Bug fixes

  • Fixed an issue where Arts & Crafters would teleport you to the wrong place in Juniper Hills
