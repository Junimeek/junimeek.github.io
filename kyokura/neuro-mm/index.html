Neuro-sama Song Pack

The Neuro-sama Song Pack is a mod for Hatsune Miku: Project DIVA MegaMix+ that aims to add all six official Neuro-sama original songs and covers to the game. Currently, LIFE and BOOM are the only songs that has been charted and implemented, with work now starting for Chinatown Blues.

This mod is a collaboration between Junimeek and Sunk.

This mod should be compatible with all platforms that Project DIVA MegaMix+ is available on. DIVA Mod Loader (DML) version 0.0.16 (or any version newer than it) is required to play this mod.

DivaModManager Install

Any front-end mod loader should work to play this mod. Enomoto's DivaModManager is the front-end that I personally use, so there shouldn't be any issues when using it. Alternatively, d4m and Rust4Diva should work too, though I don't test compatibility using those.

If you're using DivaModManager or Rust4Diva, you can click this button to initiate the mod download:
