Neuro-sama Song Pack

The Neuro-sama Song Pack is a mod for Hatsune Miku: Project DIVA MegaMix+ that aims to add all four Neuro-sama singles to the game. Currently, BOOM is the only song that has been fully charted and implemented, with some work currently being done on Seishun Complex.

This mod should be compatable with all platforms that Project DIVA MegaMix+ is available on. Enomoto's DivaModManager is required to play this mod, and Mega Mix Thumbnail Manager is recommended.
