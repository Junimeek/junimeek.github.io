VIA Pocket Schedule Errors

Notice: This page is quite outdated by now; some new errors have popped up, and other things have been fixed. This will be updated sometime later, probably late September.

Some of the official VIA schedules have errors. The vast majority of them are minor visual errors and inconsistencies (most of which won't be listed here), but some errors convey inaccurate information. So, I have created this list of errors that you may want to be aware of below.

Clicking the links will open the latest version of each schedule on VIA's website.
