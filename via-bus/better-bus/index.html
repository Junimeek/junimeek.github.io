VIA Better Bus

This page was last updated on April 28, 2026 to reflect the latest service proposals.

The Better Bus Plan is part of VIA Metropolitan Transit's Keep San Antonio Moving program that aims to increase system-wide bus frequencies to 30 minutes or less and modify all bus routes.

Below are a set of tables that list all changes that are planned to take place over the next five years organized by route type. Both current routes and brand new routes are detailed. An interactive map with the Board-adopted system (as of April 22, 2025) can be found here.

This page is extremely long, so feel free to use the table of contents below to skip to a specific piece of information.

VIA Rapid Routes

VIA Rapid (or Advanced Rapid Transit) will be San Antonio's second Bus Rapid Transit system (100 Prímo Fredericksburg and former 101 Prímo UTSA / Leon Valley were technically considered BRT lines until late 2017). The rapid transit system will feature a dedicated bus lane in either the center or the sides of the road, traffic signal priority, high frequencies, and bus stations with kiosks for off-board fare collection.

Early on, the Keep San Antonio Moving site mentioned that we would be getting two ART lines in addition to the Green and Silver line, though it is currently unknown whether we will still be getting them and/or what corridors VIA would consider for rapid transit.
